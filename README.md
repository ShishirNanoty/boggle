# boggle
Command line boggle

## usage

This is an boggle game for the command line.  

```
$ python boggle.py 
a e o e r
l o t e t
e d r a e
s q s m s
s a l p o
Type 'done!' when finished or 'board!' to reprint board.
Enter word: lot
Enter word: sqs
Enter word: zzz
Enter word: board!
a e o e r
l o t e t
e d r a e
s q s m s
s a l p o
Enter word: done!
Here are your results:
lot 3
sqs not in dictionary
zzz not on board
Score:  3
```
